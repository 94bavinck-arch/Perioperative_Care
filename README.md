# 수술 후 안내 QR

정형외과 병동에서 종이 안내문을 대신해 사용할 수 있는 정적 웹 안내문 프로토타입입니다.

## 흐름

1. 간호사가 bedside에서 병실, 환자 이름, 마취 종류, 병동 return 시간을 입력합니다.
2. `QR 생성`을 누릅니다.
3. 환자나 보호자가 휴대폰 카메라로 QR을 스캔합니다.
4. 환자용 웹 안내문에서 물 가능 시간, 식사 가능 시간, 수술 후 안내, FAQ를 확인합니다.

## 개인정보와 저장 방식

- 서버에 환자별 데이터를 저장하지 않습니다.
- 입력값은 QR 주소의 `#` 뒤에 들어갑니다.
- QR 생성은 프로젝트에 포함된 MIT 라이선스 `qrcode-generator`로 브라우저 안에서 처리됩니다.
- 환자 이름과 병실은 종이 안내문 수준의 식별 정보로만 사용하도록 설계했습니다.
- 생년월일, 등록번호, 진단명, 구체적인 수술명은 넣지 않는 것을 권장합니다.

## 실행

이 폴더에서 `index.html`을 열면 됩니다. 실제 병동에서 쓰려면 병원에서 허용한 정적 웹주소에 이 파일들을 올린 뒤, 간호사용 화면의 `웹페이지 주소`가 그 주소를 가리키게 설정하세요.

## 내용 수정

안내 문구는 `app.js` 상단의 `CARE_GUIDES`와 `OPTION_GUIDES`에서 수정할 수 있습니다. Spinal 안내는 병동과 마취과의 공식 문구로 검토 후 조정하는 것을 권장합니다.
